# 个人成长仪表板

一个功能完整的个人成长追踪工具，支持微信小程序和Web版本，帮助用户记录和管理个人成长的各个方面。

## 🌟 最新功能

### 🎊 年度总结功能
- **智能关键词分析**: 自动提取全年记录中的高频词汇，生成个性化关键词云
- **全面数据统计**: 汇总全年的记录天数、运动时长、学习时长、阅读书籍等核心数据
- **成长轨迹回顾**: 按月展示成长里程碑，智能识别每月的主要活动类型
- **成就系统**: 6种不同类型的成就徽章，根据实际数据判断解锁状态
- **个性化寄语**: 基于用户数据生成专属的年度成长寄语
- **12月31日自动提醒**: 在每年最后一天自动弹出生成年度总结的提醒
- **一键导出**: 支持将年度总结导出为高清图片，方便分享和保存

- **导出**: html2canvas
- **存储**: LocalStorage
- **响应式**: CSS Grid + Flexbox

### 微信小程序版本
- **框架**: 微信小程序原生开发
- **样式**: WXSS + Flexbox
- **数据存储**: 微信小程序云开发

## 快速开始

### Web版本部署
1. 克隆项目到本地
```bash
git clone [项目地址]
cd 个人成长计划
```

2. 启动本地服务器
```bash
# 使用Python
python -m http.server 8000

# 或使用Node.js
npx serve .

# 或使用PHP
php -S localhost:8000
```

3. 打开浏览器访问 `http://localhost:8000`

### 微信小程序部署
1. 在微信开发者工具中导入项目
2. 配置云开发环境
3. 上传代码并发布

## 使用指南

### 年度总结功能使用方法
1. **访问年度总结**: 在数据面板页面点击"年度总结"按钮
2. **选择年份**: 使用年份选择器选择要生成总结的年份
3. **生成总结**: 点击"生成年度总结"按钮，系统将自动分析该年度的所有数据
4. **查看内容**: 总结包含关键词分析、数据统计、成就展示、成长轨迹等内容
5. **导出分享**: 点击"导出图片"按钮，将年度总结保存为图片格式

### 年度总结内容说明
- **关键词分析**: 基于全年记录内容提取的高频词汇，反映年度关注重点
- **数据统计**: 包括记录天数、运动时长、学习时长、阅读书籍数量、平均情绪分数
- **成就系统**: 6种成就类型（坚持记录、运动达人、学习之星、阅读爱好者、情绪管理、全面发展）
- **成长轨迹**: 按月展示的成长里程碑，智能识别每月主要活动
- **个性化寄语**: 根据用户数据生成的专属年度总结寄语

## 使用说明

### 数据记录
1. 点击底部导航栏选择对应模块
2. 点击"+"按钮添加新记录
3. 填写相关信息并保存
4. 支持添加图片作为记录附件

### 数据查看
1. 在各模块页面查看历史记录
2. 点击记录进行编辑或删除
3. 在统计页面查看数据汇总

### 数据备份
1. 在首页点击备份按钮
2. 数据将自动备份到云端
3. 可随时恢复备份数据

## 注意事项

- 首次使用需要授权云开发环境
- 图片上传限制为3张
- 建议定期备份重要数据
- 确保网络连接正常以使用云功能

## 开发说明

### 云函数部署
项目包含两个云函数：
- `backupData`: 数据备份功能
- `restoreData`: 数据恢复功能

需要在微信开发者工具中上传并部署这些云函数。

### 图片资源
项目需要以下图片资源（可通过图标库获取）：
- TabBar图标（激活/未激活状态）
- 功能图标（备份、恢复、导出等）
- 空状态图标
- 操作按钮图标
